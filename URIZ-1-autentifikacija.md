# URIZ-1: Autentifikacija i korisnički profil

## Opis
Ovaj dokument prati implementaciju funkcionalnosti autentifikacije i upravljanja korisničkim profilom u okviru aplikacije UniBudget (TroškoviApp), u skladu sa funkcionalnim zahtevima FZ1-FZ3 iz projektne dokumentacije.

## Obuhvaćene funkcionalnosti
- Registracija novog korisnika (email + lozinka)
- Prijava postojećeg korisnika
- Upravljanje sesijom korisnika (login/logout)
- Prikaz i izmena osnovnih podataka korisničkog profila

## Tehnologije
- **Frontend:** React Native (Expo, TypeScript)
- **Backend / Auth:** Supabase Auth
- **Baza podataka:** PostgreSQL (Supabase)

## Status
Zadatak: URIZ-1
